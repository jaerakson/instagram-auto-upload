# 인스타그램 게시글·태그 최적화

## 작성 절차

### 1. 입력 자료 분석
두 파일을 읽는다:
- `_workspace/01_trend_analyst_report.md` — 인기 캡션 패턴, 해시태그
- `_workspace/02_image_creator_output.md` — 이미지 프롬프트, 생성 의도

이미지 프롬프트에서 핵심 주제와 분위기를 파악한다.

### 2. 캡션 작성

**구조:**
```
[훅 — 스크롤 멈추게 하는 첫 줄]

[본문 — 이미지와 연결된 스토리/감상, 2~3줄]

[CTA — 행동 유도]

.
.
.

[해시태그 블록]
```

**훅 패턴 (첫 줄):**
- 질문형: "이 도시에 살고 싶다면? 👇"
- 선언형: "AI가 그린 미래는 이런 모습이다"
- 감성형: "네온 불빛 아래, 시간이 멈춘 순간"
- 도발형: "이게 AI가 만든 사진이라고?"

**CTA 패턴:**
- "마음에 들면 ❤️ 저장해두세요"
- "어떤 스타일이 보고 싶은지 댓글로 알려주세요"
- "AI 아트 더 보고 싶다면 팔로우 🔔"

**기본 언어:** 한국어. 영어 표현은 자연스러운 경우에만 혼용.

### 3. 해시태그 전략

**총 20~25개, 3단계 구성:**

| 카테고리 | 게시물 수 범위 | 개수 | 예시 |
|---------|-------------|------|------|
| 대중적 | 100만+ | 5~7개 | #AIart #AI #digitalart |
| 중간 | 10만~100만 | 8~10개 | #AIartwork #AIgenerated #cyberpunkcity |
| 틈새 | 1만~10만 | 7~8개 | #nanobananapro #AIportraitart #사이버펑크아트 |

**필수 포함 해시태그:**
- #AIart #AI아트 #AIgenerated
- 이미지 스타일 관련 태그
- 한국어 태그 3~5개

### 4. 결과 저장
`_workspace/03_content_writer_post.md`에 저장:

```
## 게시글 (캡션)
{완성된 캡션}

## 해시태그
{해시태그 블록}

## 전체 게시 텍스트
{캡션 + 줄바꿈 + 해시태그 — Instagram API에 전달할 최종 텍스트}

## 작성 의도
{캡션 전략 간략 설명}
```
